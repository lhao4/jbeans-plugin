# JBeans — UI 设计方案

**版本**：v1.0  
**日期**：2026-04

---

## 一、设计原则

1. **贴合 IDEA 原生风格**：使用 IntelliJ 标准色彩和组件，不显突兀
2. **Postman 交互心智**：左树右编辑，熟悉的操作习惯，零学习成本
3. **信息密度适中**：关键信息突出，不堆砌按钮和文字
4. **状态反馈清晰**：每个操作都有加载/成功/失败的明确反馈

## 三、各区域详细设计

### 3.1 顶部标题栏

```
┌─────────────────────────────────────────────────────────────────┐
│  🔌 JBeans                                     [⚙ 设置]  [↺ 刷新] │
└─────────────────────────────────────────────────────────────────┘
```

- 插件名称 + 图标（左）
- 设置按钮：点击打开插件配置弹窗
- 刷新按钮：重新扫描进程和接口

**下拉菜单展开时**：

### 3.3 左侧方法树

```
┌──────────────────────────┐
│  🔍  搜索方法、类名...     │  ← 搜索框，实时过滤
├──────────────────────────┤
│  🏷 Dubbo 接口        [3] │  ← 分类折叠头，数字=方法数量
│    OrderService           │
│    ▸ createOrder          │  ← 方法名，选中高亮
│    ▸ cancelOrder          │
│    PayService             │
│    ▸ pay           ★      │  ← ★ 表示已收藏
│                           │
│  🌿 Spring Bean       [8] │
│    OrderServiceImpl       │
│    ▸ createOrder          │
│    ▸ calculateDiscount 🔒 │  ← 🔒 表示 private 方法（需开启开关）
│                           │
│  🌐 HTTP 接口         [5] │
│    POST /api/order/create │
│    GET  /api/order/{id}   │
│    DELETE /api/order/{id} │
│                           │
│  🔧 静态方法          [2] │
│    IdCardUtils            │
│    ▸ validate             │
│    ▸ mask                 │
│                           │
│  [+ 添加静态类]           │
└──────────────────────────┘
```

**交互细节**：
- 单击方法名：右侧加载该方法详情 + 参数模板
- 双击方法名：展开查看参数类型预览
- 右键菜单：收藏 / 查看源码 / 复制方法签名
- 搜索时：树结构折叠，仅展示匹配结果，高亮关键词

**搜索结果样式**：
```
匹配"createOrder"后：

  🔍 createOrder                              ← 搜索词高亮
  ──────────────────────────────────────
  createOrder                                 ← 方法名
  OrderServiceImpl · com.example.service      ← 类名+包名
  ──────────────────────────────────────
  createOrder
  OrderController · com.example.controller
```

## 五、色彩与图标规范

### 5.1 颜色规范

严格跟随 IDEA 主题配色，使用 JBColor 适配亮/暗主题：

| 用途 | 亮色主题 | 暗色主题 |
|------|---------|---------|
| 成功状态 | #2E7D32 | #4CAF50 |
| 失败状态 | #C62828 | #F44336 |
| 警告状态 | #E65100 | #FF9800 |
| 连接中 | #1565C0 | #42A5F5 |
| 禁用/灰色 | #9E9E9E | #757575 |
| 高亮选中 | IDEA 默认选中色 | IDEA 默认选中色 |

### 5.2 图标规范

| 位置 | 图标 | 说明 |
|------|------|------|
| 插件入口 | 自定义 SVG（豆子+闪电） | Tool Window 图标 |
| Dubbo 接口 | 🏷 标签样式图标 | 分类标识 |
| Spring Bean | 🌿 绿叶图标 | Spring 官方色调 |
| HTTP 接口 | 🌐 地球图标 | HTTP 标识 |
| 静态方法 | 🔧 扳手图标 | 工具类标识 |
| private 方法 | 🔒 锁图标 | 访问限制提示 |
| 收藏 | ★ / ☆ | 收藏状态 |

### 5.3 JSON 高亮配色

跟随 IDEA 编辑器主题，标准 JSON 语法高亮：
- 字符串值：绿色
- 数字值：蓝色
- 布尔/null：紫色
- key 名：白色/黑色（跟随主题）
- 括号：默认前景色

## 六、交互动效

| 场景 | 动效 |
|------|------|
| 点击 Invoke | 按钮内旋转 loading 图标，禁用状态 |
| 连接进程中 | 状态点闪烁动画 |
| 搜索过滤 | 列表渐入渐出（200ms） |
| 结果展示 | 从顶部渐入（150ms） |
| 加载接口列表 | 骨架屏占位，逐行渐入 |
| 错误提示 | 从右侧滑入 Toast（3s 自动消失） |
